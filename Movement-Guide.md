### Controls
#### Dominant Hand
Using your dominant hand to aim and shoot is critical to reduce the time it takes to naturally respond to threats in the AO, significantly decreasing TTK (Time-To-Kill). 

For a right-handed player to use this, you will first need to get used to using Southpaw. It swaps aim and movement controls, which may be disorienting at first. After getting it down, you’ll want to make the following changes:
* Map “Aim Weapon” to R2/RT
* Map “Shoot Weapon” to R1/RB
* Map “Use Lethal Equipment” to L1/LB
* Map “Use Tactical Equipment” to L2/LT

For a left-handed player–assuming they haven’t been using Southpaw before– make the following changes:
* Map “Aim Weapon” to L2/LB

This change in controls makes it easy to shoot with one hand, while making it easy on the other to control recoil and make adjustments to the camera. This type of setup is called “Dominant One-Handed Gunslinger”, and the left-handed version of it can be found in the button presets, BUT custom needs to be used to combine the two.

#### Tactical Buttoning
Using the tactical-type preset actually makes it easier for you to slide-cancel, slide-jump, etc. And, instead of taking your hand off the right analog to slide, you can do so seamlessly.

For right-handed people, make the following changes:
* Map “Change Stance” to LSC (Left Stick Click)

For left-handed people, make the following changes:
* Map “Change Stance” to RSC (Right Stick Click)

This change in controls makes it easy to go into the different stances, allowing for easy versatility, which is essential for movement demons.

### Aim Response Curve
This setting is usually untouched by any casual player because it is just fine for most things. Not for a demon. When set to Default, the aim starts slower than the aim sensitivity and then goes up to reach it. This can cause some problems and actually increase the amount of time before you start shooting. For this not to happen, we need to change that setting to something more usable.

To adjust this, use the following menu path:
1. Navigate to **Controller**
2. Go to Submenu **Aiming**
3. Click into **Aiming Advanced Settings**
4. Click **Aim Response Curve Type**
5. Set to **Dynamic**
6. Change its slope scale/strength to **0.9** or similar.

With this type of curve setting, you actually start faster than regular and slow down to meet the aiming sensitivity. This is crucial for quick movements that require you to snap and turn around instantly.
